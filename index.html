[1] O SANTUÁRIO F-DROID (BYPASS PLAY STORE)

A Google Play Store censura o Termux e impede atualizações. A solução tática é usar o repositório F-Droid, focado em software livre.

1. Acesse f-droid.org pelo navegador do celular.
2. Baixe o APK e instale a loja.
3. Pesquise por Termux Terminal Emulator with Packages e instale.

[2] CALIBRAÇÃO DA ENGINE

Ao abrir o Termux pela primeira vez, você precisa atualizar os repositórios base e conceder acesso ao armazenamento interno:

pkg update && pkg upgrade -y
termux-setup-storage

[3] A CAMADA PROOT (LINUX NO BOLSO)

Para rodar distribuições completas sem depender de privilégios de Root, utilizamos o pacote de orquestração PRoot:

pkg install proot-distro -y
proot-distro list

[4] ESCOLHA SUA ARQUITETURA

UBUNTU (O Peso Pesado): Compatibilidade máxima com frameworks corporativos. Exige mais recursos do aparelho.

proot-distro install ubuntu
proot-distro login ubuntu

ALPINE LINUX (O Drone Invisível): Arquitetura minimalista e agressiva. Consome menos de 5MB iniciais. Perfeito para execução rápida de scripts.

proot-distro install alpine
proot-distro login alpine

ARCH LINUX (Vanguarda): Doutrina Rolling Release. Controle absoluto sobre os pacotes para operadores avançados.
